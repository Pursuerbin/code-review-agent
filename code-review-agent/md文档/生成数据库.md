# 数据库创建指南

## 一、PostgreSQL 数据库设置

### 1. 创建数据库

打开 PostgreSQL 命令行或使用 pgAdmin 执行以下 SQL：

```sql
-- 创建数据库
CREATE DATABASE code_review_db 
  WITH 
  OWNER = postgres
  ENCODING = 'UTF8'
  LC_COLLATE = 'zh_CN.UTF-8'
  LC_CTYPE = 'zh_CN.UTF-8'
  TABLESPACE = pg_default
  CONNECTION LIMIT = -1;
```

### 2. 创建表结构

连接到 `code_review_db` 后执行以下 SQL：

```sql
-- 创建 review_task 表
CREATE TABLE IF NOT EXISTS review_task (
    id BIGSERIAL PRIMARY KEY,
    code_content TEXT NOT NULL,
    lang VARCHAR(50) DEFAULT 'java',
    status INTEGER DEFAULT 0,
    result_summary TEXT,
    task_type VARCHAR(50) DEFAULT 'REVIEW',
    extra_context TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 创建索引
CREATE INDEX IF NOT EXISTS idx_review_task_status ON review_task(status);
CREATE INDEX IF NOT EXISTS idx_review_task_type ON review_task(task_type);
CREATE INDEX IF NOT EXISTS idx_review_task_created_at ON review_task(created_at);
```

### 3. 表字段说明

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `id` | BIGSERIAL | 主键，自增 |
| `code_content` | TEXT | 待审查的代码内容 |
| `lang` | VARCHAR(50) | 编程语言（java/python/javascript等） |
| `status` | INTEGER | 状态：0-待处理，1-处理中，2-成功，3-失败 |
| `result_summary` | TEXT | 审查结果摘要 |
| `task_type` | VARCHAR(50) | 任务类型：REVIEW/CodeGenerationAgent等 |
| `extra_context` | TEXT | 额外上下文信息 |
| `created_at` | TIMESTAMP | 创建时间 |
| `updated_at` | TIMESTAMP | 更新时间 |

## 二、MySQL 数据库设置（可选）

### 1. 创建数据库

```sql
CREATE DATABASE IF NOT EXISTS code_review_db 
  DEFAULT CHARACTER SET utf8mb4 
  DEFAULT COLLATE utf8mb4_unicode_ci;
```

### 2. 创建表结构

```sql
USE code_review_db;

CREATE TABLE IF NOT EXISTS review_task (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    code_content TEXT NOT NULL,
    lang VARCHAR(50) DEFAULT 'java',
    status INT DEFAULT 0,
    result_summary TEXT,
    task_type VARCHAR(50) DEFAULT 'REVIEW',
    extra_context TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_status (status),
    INDEX idx_task_type (task_type),
    INDEX idx_created_at (created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

## 三、配置文件修改

修改 `application.yml` 中的数据库连接配置：

```yaml
spring:
  datasource:
    url: jdbc:postgresql://localhost:5432/code_review_db
    username: postgres
    password: your_password
    driver-class-name: org.postgresql.Driver
```

## 五、注意事项

1. **权限设置**：确保数据库用户有足够的权限（CREATE, INSERT, UPDATE, DELETE）
2. **编码设置**：建议使用 UTF-8/UTF8MB4 编码支持中文
3. **连接池配置**：生产环境建议调整 HikariCP 连接池参数
4. **备份策略**：定期备份数据库，建议使用 PostgreSQL 的 `pg_dump` 或 MySQL 的 `mysqldump`
